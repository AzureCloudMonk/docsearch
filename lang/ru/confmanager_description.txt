Плагин DocSearch позволяет выполнять поиск среди загруженных документов.
Он интегрирован в дефолтный поиск ДокуВики.
Просто заполните строку поиска и начните поиск.

DocSearch требуются внешние программы (компоненты) для преобразования специфичных типов файлов в текстовые данные.
Разместите расширение файла в ключевой (Key) колонке и программу преобразования в правой.
При обнаружении файла с указанным расширением будет использована соответствующая программа.

DocSearch должен сообщить программе преобразования путь к преобразуемому файлу и целевой путь для записи текстового файла.
Это реализуется использованием меток %in% для файла- источника и %out% для файла-получателя.
Сгенерированный текст должен быть в кодировке UTF-8.

Пример:
  /usr/bin/pdftotext -enc UTF-8 %in% %out%
Подробнее смотри в [[doku>plugin:docsearch|documentation]].
